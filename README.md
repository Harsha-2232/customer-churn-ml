# Customer Churn Prediction API

This project predicts whether a telecom customer will churn using a Machine Learning model deployed with FastAPI.

## Tech Stack

- Python
- Pandas
- Scikit-learn
- FastAPI
- Uvicorn

└── requirements.txt
